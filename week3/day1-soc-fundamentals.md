# Week 3 – Day 1: SOC Fundamentals

## Focus
Introduction to Security Operations Centers (SOC) and the blue team mindset used to detect, analyze, and respond to security incidents.

---

## Topics Covered
- Purpose and function of a SOC
- Blue team vs red team roles
- Incident detection and response workflow
- Importance of logs and monitoring
- Overview of SIEM tools and alerts

---

## Key Concepts
- **SOC:** Centralized team responsible for monitoring and responding to security events
- **Blue Team:** Defensive security professionals
- **Alert:** Notification of potentially malicious activity
- **Incident:** Confirmed security event requiring response
- **SIEM:** Tool used to collect and analyze logs centrally

## Key Takeaways
SOC analysts focus on visibility, detection, and response. Logs, alerts, and monitoring systems are essential for identifying threats and minimizing impact.
